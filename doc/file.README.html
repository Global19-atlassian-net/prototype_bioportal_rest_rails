prototypebioportalrest_rails

This is a prototype BioPortal web service using rails-api as the framework

Getting started

  1. Install ruby 1.9.3
  2. gem install bundler
  3. git clone https://github.com/ncbo/prototypebioportalrest_rails.git
  4. bundle
  5. rails server
